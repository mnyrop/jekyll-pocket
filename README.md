# jekyll_pocket

jekyll hook plugin for building a fully portable site 🎣 💾👝

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'jekyll_pocket'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install jekyll_pocket
